# Sidebar Collapse Design

## Goal

左侧侧边栏支持手动折叠，并在下次打开页面时恢复上次状态。

## Approach

沿用当前项目已有的 `sls-log-inspector:prefs` 偏好存储，把 `sidebarCollapsed` 加入 `localStorage`。页面初始化时读取该状态，点击侧边栏按钮时更新状态、写回偏好并切换布局 class。

## UI Behavior

- 在侧边栏品牌区域右侧新增一个折叠按钮。
- 展开状态下保持现有 250px / 230px 侧边栏宽度。
- 折叠状态下侧边栏缩窄，仅保留品牌标识和展开按钮。
- 拖拽上传区、索引字段区和品牌文案在折叠状态隐藏。
- 移动端仍保持单列布局，折叠状态只收起侧边栏内容，不影响主工作区。

## Verification

默认不运行测试命令。完成后通过静态检查、语法检查、本地页面冒烟和远程服务健康检查验证。
